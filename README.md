# Automação de Testes com Playwright

Projeto desenvolvido para prática de automação de testes utilizando Playwright.

## Objetivo
Praticar testes automatizados E2E, validando cenários positivos e negativos.

## Estrutura do projeto

- tests/basic: testes simples para aprendizado inicial
- tests/intermediate: cenários mais completos simulando situações reais

## Cenários testados

- Login com sucesso
- Login inválido
- Validação de erro
- Interação com elementos da página

## Tecnologias utilizadas

- Playwright
- JavaScript

## Como executar o projeto

```bash
npm install
npx playwright test
